## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/silvaMatheus/matheusdeveloper.com.git
```

2. Then go to the project's folder:

```sh
cd matheusdeveloper.com
```

3. Install all dependencies:

```sh
pnpm install
```

4. Run locally:

```sh
pnpm run dev
```
